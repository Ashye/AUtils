# AUtils
###### 功能：

+ 联网：使用了 [Retrofit](https://github.com/square/retrofit)
+ 图片加载：使用了 [Android-Universal-Image-Loader](https://github.com/nostra13/Android-Universal-Image-Loader)
+ 本地储存：使用了 Android 自带的 **SharePreferences**
